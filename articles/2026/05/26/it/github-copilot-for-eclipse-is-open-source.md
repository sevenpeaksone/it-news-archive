# GitHub Copilot for Eclipse is open source

## 핵심 내용
GitHub가 Eclipse용 GitHub Copilot 확장 코드를 MIT 라이선스로 공개했습니다. Eclipse 기반 개발 환경을 쓰는 팀은 확장 동작을 직접 검토하거나 내부 정책에 맞춰 수정할 수 있는 여지가 커졌습니다.
