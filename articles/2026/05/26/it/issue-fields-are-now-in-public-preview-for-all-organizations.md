# Issue fields are now in public preview for all organizations

## 핵심 내용
GitHub 조직에서 이슈에 Priority, Effort 같은 구조화된 필드를 붙이는 기능이 공개 프리뷰로 열렸습니다. 단순 라벨보다 일관된 업무 분류와 우선순위 관리를 하려는 팀에 유용한 변화입니다.
